# zillizcloud_api_keys (Data Source)

Lists all Customized API keys in the organization.

**Important:** This data source requires an Org Owner API key. A project-scoped key
cannot list API keys (error 96041).

## Example Usage

```terraform
data "zillizcloud_api_keys" "all" {}

output "key_names" {
  value = [for k in data.zillizcloud_api_keys.all.api_keys : k.name]
}

# Find keys scoped to a specific project
output "project_keys" {
  value = [
    for k in data.zillizcloud_api_keys.all.api_keys : k
    if length([for p in k.projects : p if p.project_id == "proj-xxxx"]) > 0
  ]
}
```
